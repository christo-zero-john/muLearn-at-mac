Step 1: Select a student

Step 2: Select task to be added for the selected student

Step 3: Click on ADD to add the selected task to the task list of selected student.

Step 4: Repeat steps 1, 2, and 3 and add all students and their corresponding completed tasks to be submitted. You can see added data to be submitted in the table which will appear below as you add new data

Step 5: Click on submit button and submit data.

    Note:
  • To view already completed tasks of a student, first select the name of the student from the input and click on view completed tasks button, which will appear then.
  • The ADD and submit buttons are disabled by default.
  • The ADD button gets enabled when valid students and hashtag is selected from corresponding input fields.
  • The submit button gets enabled when atleast one student is added.
  • After Clicking the submit button (that is after submitting the data), there will be a message confirming your submission was success. Refresh the page after 10 seconds when that message is displayed and do not add any data before refreshing.
  • Try to submit all data atonce and avoid submitting multiple times.
  • To view already completed tasks of a students, enter the muid and click on view completed tasks buton which will appear.
